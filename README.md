# 🏥MedTalk-Platform-Konsultasi-Online
Tugas Project Mahasiswa: Mendesain Sistem Telemedicine Sederhana untuk Klinik Kecil.

## 1. Pendahuluan
MedTalk adalah platform telemedicine yang dirancang untuk membantu klinik kecil mengelola konsultasi pasien secara daring, mulai dari pendaftaran hingga pemberian resep digital.

## 2. Diagram Arsitektur Sistem (Nilai: 40%)
Sistem ini menggunakan arsitektur **Three-Tier**:
* **Frontend:** Antarmuka Pasien & Dokter (Web-based).
* **Backend:** Server pengolah data (API Services).
* **Database:** MySQL (Penyimpanan Rekam Medis).

## 3. Komponen Utama (Nilai: 30%)
* Modul Booking Jadwal Real-time.
* Fitur Video Call Konsultasi.
* E-Prescription (Resep Digital).

## 4. Alur Pasien
Pendaftaran -> Pilih Dokter -> Booking -> Konsultasi Video -> Terima Resep.
